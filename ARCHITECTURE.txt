STORY WEAVER DATA FLOW:

   Frontend                          Gemini API                    Backend
   ========                          ==========                    =======

1. User Input
   ├─ Language: "French"
   ├─ Words: "gato, leche, rápido"
   └─ Click "Weave Story"

2. Generate Story
                    ──────────────────────────────────────►
                    gemini-3.1-flash-lite-preview
                    (Generate story JSON)
                    ◄──────────────────────────────────────
   Display Story
   (with highlights)

3. Generate Audio (Optional)
                    ──────────────────────────────────────►
                    gemini-2.5-flash-preview-tts
                    (Generate audio base64)
                    ◄──────────────────────────────────────
   Play Audio
   (with seek bar)

4. Save Story
   (Base64 audio)
                                                    POST /stories
                                    ──────────────────────────────►
                                    Validate & Process:
                                    ├─ Validate vocabulary JSON
                                    ├─ Decode base64 audio
                                    ├─ Save audio to /uploads/
                                    ├─ Create Story model
                                    └─ Commit to database
                                    ◄──────────────────────────────
   Show Success

5. List Saved Stories
                                                    GET /stories
                                    ──────────────────────────────►
                                    Query all stories
                                    from database
                                    ◄──────────────────────────────
   Display List
   in Sidebar

6. Load Saved Story
                                                GET /stories/{id}
                                    ──────────────────────────────►
                                    Query story by ID,
                                    parse vocabulary JSON,
                                    return full data
                                    ◄──────────────────────────────
   Display Story
   + Audio Link

7. Delete Story
                                              DELETE /stories/{id}
                                    ──────────────────────────────►
                                    ├─ Delete from database
                                    └─ Delete audio file
                                    ◄──────────────────────────────
   Reload List

VOCABULARY TABLE STRUCTURE:

┌────────────────────────────────────────────────────────────────────────────┐
│ Original Word │ Meaning in French │ Equivalent Word in English             │
├────────────────────────────────────────────────────────────────────────────┤
│ gato           │ gato               │ cat, a furry animal companion         │
│ leche          │ lait               │ milk, white drink from cow            │
│ rápido         │ rapide             │ fast, speedy movement                 │
└────────────────────────────────────────────────────────────────────────────┘

Each row is a VocabWord:
{
  "word": "gato",
  "meaning_in_target": "gato",
  "equivalent_in_english": "cat, a furry animal companion"
}
